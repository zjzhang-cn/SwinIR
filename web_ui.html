💡 使用提示
  • 上传图像后点击「开始增强」，等待处理完成即可查看结果
  • 大分辨率图像建议填写分块大小（如 400），避免显存不足
  • 如需使用其他任务（经典 SR、轻量 SR 等），请在 /tasks 接口查看，然后在 /predict 中传参
  • 服务默认监听 0.0.0.0:8000，可通过 /docs 访问 Swagger API 文档
  • Mac Apple Silicon 自动使用 MPS 加速，NVIDIA GPU 使用 CUDA 加速
